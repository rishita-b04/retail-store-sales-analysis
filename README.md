# retail store sales analysis dashboard
## Project Overview
This project analyzes retail store sales data to identify sales trends, profitability, customer segments, and regional performance using Python.

## Objectives
- Analyze sales and profit
- Identify top-performing categories
- Analyze customer segments
- Generate business insights
- Build an interactive dashboard

## Tech Stack
- Python
- Pandas
- NumPy
- Plotly
- Jupyter Notebook

## Dataset
Sample Superstore Dataset

## Key Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Interactive Visualizations
- Sales Dashboard
- Business Insights
- Recommendations

## Dashboard Includes
- Sales by Category
- Profit by Category
- Sales by Region
- Sales by Segment
- Sales vs Profit
- Correlation Heatmap

## Business Insights
- Technology category generated the highest sales and profit.
- Consumer segment contributed the highest revenue.
- West region achieved the highest sales.
- Large discounts negatively affected profitability.
